# ChatLog

## 功能介绍

这是一个Chrome浏览器插件（Manifest V3），用于在ChatGPT网页中提升长对话的浏览效率。主要功能包括：

1. **提取用户历史对话**：只提取用户发送的历史对话内容
2. **右侧列表展示**：在页面右侧以列表形式展示历史对话，每条只显示一行摘要
3. **关键词搜索**：支持通过关键词搜索历史输入
4. **快速跳转**：点击某一条记录后快速跳转到页面中的对应位置
5. **高亮效果**：跳转后对目标内容进行短暂高亮
6. **点击跳转提示**：在列表顶部显示明确的操作提示
7. **文件消息处理**：优化处理带有文件附件的消息，显示[文件]标记
8. **可拖动调整大小**：支持左右拖动调整历史对话列表的宽度

## 技术实现

### 核心架构

- **适配器模式**：使用PlatformAdapter接口定义平台适配方法，当前实现ChatGPTAdapter
- **实时监听**：使用MutationObserver监听页面变化，实时更新消息列表
- **防抖处理**：避免频繁更新导致性能问题
- **响应式设计**：适配不同屏幕尺寸

### 关键特性

1. **DOM结构稳定性**：使用data属性（如data-message-author-role="user"）识别用户消息，不依赖className
2. **可靠的跳转功能**：基于消息ID进行定位，支持多种查找方式
3. **扩展能力**：通过适配器层支持其他平台
4. **性能优化**：防抖处理、高效DOM操作
5. **文件消息处理**：智能识别文件附件，显示[文件]标记和文件名（如果可提取）
6. **可拖动调整大小**：支持通过拖动分隔条调整历史对话列表宽度
7. **响应式设计**：消息内容自动适应列表宽度，保持单行显示
8. **搜索状态保持**：页面变化时保持搜索过滤状态，不会自动跳回到显示所有消息

### 数据结构

每条消息的数据结构包含：
- id：消息唯一标识符
- content：完整内容（包含[文件]标记，如果有文件附件）
- preview：预览文本（最多100字符）
- index：消息索引
- element：消息DOM元素

## 安装说明

1. 克隆或下载本项目到本地
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目目录
6. 插件将自动安装并在ChatGPT页面生效

## 后续扩展

1. **支持其他平台**：可通过创建新的适配器类支持其他AI聊天平台
2. **本地存储**：可添加本地存储功能，保存历史对话记录
3. **更多搜索选项**：支持高级搜索，如按时间、关键词权重等
4. **自定义样式**：添加用户可配置的样式选项

## 注意事项

- 插件仅在ChatGPT网页（https://chatgpt.com/*）上生效
- 所有数据仅保存在本地，不上传服务器
- 插件会实时监听页面变化，可能会对性能有轻微影响

## 开发说明

### 主要文件

- `manifest.json`：插件配置文件
- `content.js`：核心功能实现，包含：
  - 适配器模式实现
  - 消息提取和处理逻辑
  - 文件附件识别
  - 可拖动调整大小功能
- `style.css`：样式文件，包含：
  - 历史对话列表样式
  - 可拖动分隔条样式
  - 高亮动画效果
- `icon16.png`、`icon48.png`、`icon128.png`：插件图标
